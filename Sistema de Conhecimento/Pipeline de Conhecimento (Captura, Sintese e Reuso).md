# Conteudo

Pipeline aplicado ao vault:
1. Captura: registrar ideia/fonte com contexto minimo.
2. Sintese: escrever em palavras proprias (nao colar texto cru).
3. Atomicidade: uma nota, um conceito central.
4. Conexao: criar links com notas existentes e justificar relacao.
5. Reuso: transformar cluster de notas em topicos, guias e novas explicacoes.

Heuristica de qualidade:
- Destacar menos e selecionar mais.
- Priorizar fontes com maior sinal (livro, artigo tecnico, docs oficiais).
- Criar MOC (mapa) quando houver 7+ notas relacionadas.
